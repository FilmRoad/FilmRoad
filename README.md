# FlimRoad
🔥 영화, 드라마 속 그 장소를 직접 방문할 수 있는 여행 가이드 앱

당신이 사랑한 영화·드라마의 촬영지를 한눈에

실제 촬영 장소의 위치, 정보, 설명을 제공

이제는 스크린 속 감동을 현실에서 경험하세요!
<br><br>
